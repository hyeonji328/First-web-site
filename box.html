박스 모델

태그 하나하나를 하나의 박스로 취급해서 레이아웃을 생성하거나 정렬하는 것. CSS의 모든 요소는 박스에 들어가고 이 박스를 이용해 디자인함. block과 inline이라는 크게 두 개의 박스가 있음. 화면 전체를 사용하는 태그를 block level element라고 하고, 자기 콘텐츠만큼만 쓰는 태그를 inline element라고 함. 이 둘은 display 속성의 기본값이라 CSS로 언제든 바꿀 수 있음.
